Project Deliverables for Linear Regression:
----------------------------------------------------------------------------------------------------------------------------
Do classification using data (http://archive.ics.uci.edu/ml/machine-learning-databases/iris/iris.data).
Requirement:
1. Training a model via linear regression;
2. Using the trained model to do the classification;
3. Need N-fold cross validation

------------------------------------------------------------------------------------------------------------------------------
In order to run the program, you will require to have the following files in the same folder.
below is the list of the files you will need to load in a folder.
Project1.py 
libraries.txt
iris_dataset.csv
iris_data_test.csv
-------------------------------------------------------------------------------------------------------------------------------
Description of files:
Project1.py : Comprises of the program
libraries.txt : Comprises of the dependent libraries used in the project to load before running program
iris_dataset.csv : actaul Iris Dataset
iris_data_test.csv : This a random set of data that I have selected to test the model in order to predict values of Y and compare them with actual values of Y to calculate accuracy.
-------------------------------------------------------------------------------------------------------------------------------
Steps to run the program in terminal:
1) You will need a 3.Z or higher version to run this program. I have used 3.6.5 version to run the program.
2) Go to the folder where files are present using cd command
3) You will have to load the required external libraries to run the project using the command
	pip install -r libraries.txt
4) Run the program using command prompt of project directory using command
	python Project1.py
