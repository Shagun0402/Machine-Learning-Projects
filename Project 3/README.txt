K-Means Clustering on Iris Data
-------------------------------------------------------------------------------------------------------------------
Project Deliverables:
Do clustering using data (http://archive.ics.uci.edu/ml/machine-learning-databases/iris/iris.data).
Requirement:
Fulfill the K-means clustering algorithm.

------------------------------------------------------------------------------------------
Files to be uploaded:
(1) iris_dataset.csv
(2) packages.txt
(3) Kmeans.py
(4) results.csv

Instructions to run on Command Prompt:
NOTE: This program runs on 3.X Python version. Please load all the above mentioned files into a folder.
(1) Navigate to folder that comprises of above mentioned files.
(2) Install external libraries using
	[ pip install -r packages.txt]
(3) Run the program using
	[python Kmeans.py]
(4) The final results will be saved in results.csv
